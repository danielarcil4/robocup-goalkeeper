Detailed Description

Macro definitions for GPIO pins used in the RoboCup Goalkeeper firmware project.

This header provides symbolic names for the GPIO pins assigned to various hardware components, including encoders (AS5600), motors (SKYWALKER), and the IMU (BNO055).

  • Encoder AS5600: Defines analog input pins for three encoders.
  • Motor SKYWALKER: Defines PWM output pins for signal and reverse control of three motors.
  • IMU BNO055: Defines UART TX and RX pins for IMU communication.
Note
Update these definitions if the hardware pinout changes.
