int init_motors ( void )

Initialize the motors This function initializes the brushless motors using LEDC PWM.

It sets up the GPIO pins, LEDC timers, and channels for motor control.

Returns
int
  • INIT_SUCCESS if initialization is successful
  • INIT_ERROR_MOTOR if motor initialization fails

int init_sensors ( void )

Initialize all sensors used in the system (AS5600, BNO055, ADC units).

Initialize the sensors This function initializes the AS5600 and BNO055 sensors, sets up the UART for communication, and configures the GPIO pins for the sensors.

Configures shared ADC unit, per-sensor ADC channels and other sensor-specific setup. Returns an initialization status code.

Returns
int INIT_SUCCESS on success, or an INIT_ERROR_* code on failure
