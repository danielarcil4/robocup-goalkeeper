Initialize the sensors This function initializes the AS5600 and BNO055 sensors, sets up the UART for communication, and configures the GPIO pins for the sensors.

Configures shared ADC unit, per-sensor ADC channels and other sensor-specific setup. Returns an initialization status code.

Returns
int INIT_SUCCESS on success, or an INIT_ERROR_* code on failure
