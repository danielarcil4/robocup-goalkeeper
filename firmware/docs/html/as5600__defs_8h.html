enum AS5600_ff_threshold_t

Fast filter threshold options for FF bitfield at CONF register.

For a fast step response and low noise after settling, the fast filter can be enabled

The fast filter works only if the input variation is greater than the fast filter threshold, otherwise the output response is determined only by the slow filter
