Detailed Description

Driver helpers for the AS5600 magnetic rotary sensor.

Implements initialization, register access, ADC-based angle reading and configuration helpers used by encoder tasks.

◆ AS5600_ADC_GetAngle()

float AS5600_ADC_GetAngle ( AS5600_t * as5600)

Read angle from AS5600 using the analog OUT pin (via ADC).

Get angle in degrees from the AS5600 sensor by ADC.

Returns angle in degrees [0,360) when ADC is calibrated and the OUT stage is reduced range (10%-90%). Returns -1 on error or when ADC not available.

void AS5600_BurnAngleCommand ( AS5600_t * as5600)

Perform BURN_ANGLE command to permanently program ZPOS/MPOS.

The host microcontroller can perform a permanent programming of ZPOS and MPOS with a BURN_ANGLE command.

Executes the required register write to trigger the burn command (can be used up to 3 times).
